# MootCourt AI

An AI-powered litigation simulation platform for Indian Law Students and Advocates.

## Features
- **AI Co-Counsel**: Instant legal research, precedent analysis, and case facts review.
- **Judge Simulator**: Practice courtroom arguments in front of simulated Supreme Court Justices (D.Y. Chandrachud, V.R. Krishna Iyer, P.N. Bhagwati).
- **Memorial Builder**: Automatically draft petitioner and respondent memorials.
